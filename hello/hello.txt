main 함수: 언어에서 가장 첫 번째로 실행되는 함수
printf() -> 화면에 문자열 출력해주는 함수  
함수:()가 붙은 단어 ; -> 세미콜론: 줄 끝마침(안 붙이면 컴파일 에러)
소스코드 편집-> 컴파일-> 실행 hello.c->hello.obj->hello.exe
obj와 exe는 visua studio와 윈도우 기준(OS나 컴파일러에 따라
달라질 수 있음) 서식 지정자: printf로 문자열을 출력할 때 값으로 
바뀌는 부분, printf("%s %s\n", "Hello", "1234");
서식지정자는 변수를 사용해서 값은 값 여러개를 출력하거나 출력 형태를
바꿀 때 유용함, 주석: 사람만 알아볼 수 있도록 작성하는 부분(//, /*, */)
{}는 if나 for 등의 영역을 나타냄, 들여쓰기 -> 가독성을 높여줌
C언어는 위에서 아래로 해석함(변수 선언을 한 후에 사용해야됨)
변수명 규칙: 1. 영문 문자와 숫자 사용가능 2. 대소문자 구별 
3. 문자부터 시작해야됨 4. _로 시작가능 
5. C언어 키워드(if,for,while,struct..)는 사용 X
int num3 = 7; //변수 선언하면서 값 할당(초기화)
버그: 프로그램이 의도치 않은 동작을 일으키는 것
디버거: 프로그램 내부를 보는 기능->버그 제거가 가능함(수동)
종단점(break point): 소스코드의 특정 지점에서 프로그램 실행을 멈춤
중단점 사용하기: 맨 왼쪽 클릭 또는 F9, 디버깅 시작: F5
디버깅 중지: shift + F5, 프로시저 단위 실행: F10(한 줄씩 실행)
한 단계씩 코드 실행: F11(함수진입 시에)
